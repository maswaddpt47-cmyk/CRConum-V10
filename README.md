Saisir les notes brutes
Chaque section dispose d'un champ de saisie libre pour les notes brutes de réunion.

Bonnes pratiques de saisie
Pour obtenir une reformulation de qualité optimale, il est recommandé de :
•	Écrire une idée par ligne (saut de ligne entre chaque point)
•	Formuler les actions avec un verbe d'action (éviter, contacter, valider, préparer…)
•	Ne pas chercher à rédiger — les notes brutes suffisent, l'IA reformule

✅	Exemple de bonne saisie : Bilan formation agents — 3 sessions réalisées Participation satisfaisante Relancer les agents non inscrits Préparer le rapport pour la prochaine réunion

⚠️	Éviter d'écrire un seul long paragraphe sans saut de ligne : l'IA aura du mal à séparer les idées et les actions.
